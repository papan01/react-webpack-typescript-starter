# web-app-starter

A boilerplate using React、TypeScript、Webpack、Babel、ESlint、Jest、Enzyme。

### Directory Layout
```
root-dir
├── /dist/
├── /src/
│   ├── /components/
│   │   ├── /app/
│   │   │   ├── /__tests__/
│   │   │   │   └── app.test.tsx
│   │   │   ├── /sass/
│   │   │   │   └── ...
│   │   │   └── app.tsx
│   │   └── index.d.tes
│   ├── /entry/
│   │   ├──client.js
│   ├── /sass/
│   │   ├──style.scss
│   └── /templates/
│       ├── index.html
├── babel.config.js
├── jest.confing.js
├── package.json
├── .eslintrc
├── tsconfig.json
└── webpack.config.js
```
